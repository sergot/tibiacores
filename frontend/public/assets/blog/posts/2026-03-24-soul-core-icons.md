We've added **soul core item icons** next to every creature name across TibiaCores!

## ✨ What's New

Each creature in your soul core lists and the creature search dropdown now shows its soul core sprite — the small animated icon you'd see in-game when inspecting a soul core.

![Soul core icons next to creature names](/assets/blog/images/soul-core-icons-screenshot.png)

## 🔍 Where You'll See Them

- **List detail view** — icons appear next to creature names in the soul cores table
- **Creature search** — icons show up in the autocomplete dropdown when adding a new soul core
